# Oğuzhan Özdemir — Portfolio

Personal portfolio website built with **Flutter Web**.

🌐 **Live:** [ozdemiroguz.github.io](https://ozdemiroguz.github.io)

## About

This is my interactive portfolio, designed as a mobile phone UI running in the browser. Built entirely with Flutter Web and deployed via GitHub Pages.

## Tech

- **Flutter Web** — UI framework
- **BLoC** — State management
- **Clean Architecture** — Project structure
- **GitHub Pages** — Hosting

## Features

- About, Career, Projects, Achievements screens
- CV download
- Interactive map
- Snake & Tetris games
- App Store profile link
- Turkish / English localization
